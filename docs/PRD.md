# 노션 견적서 뷰어 MVP PRD

## 🎯 핵심 정보

**목적**: 노션 데이터베이스에 작성한 견적서를 클라이언트가 별도 프로그램 없이 웹에서 바로 확인하고 PDF로 저장할 수 있게 한다.
**사용자**: 노션으로 견적서를 작성/관리하는 프리랜서·소규모 사업자(발행자)와, 발행자에게 받은 링크로 견적서를 확인하는 고객(클라이언트).

## 🚶 사용자 여정

```
[발행자 플로우]
1. 로그인 페이지
   ↓ 로그인 성공

2. 견적서 목록 페이지
   ↓ 노션 데이터베이스 동기화 / 견적서 선택

   [신규 진입] → 동기화 버튼 클릭 → 노션 데이터베이스의 견적서 목록 표시
   [기존 견적서 클릭] → 견적서 상세 페이지로 이동
   ↓

3. 견적서 상세 페이지
   ↓ 공유 링크 생성

4. 공유 링크 복사 완료 → 발행자가 클라이언트에게 링크 전달


[클라이언트 플로우]
1. 견적서 공유 페이지 (고유 링크로 직접 접속, 로그인 불필요)
   ↓ 견적서 내용 확인

   [PDF 저장 필요] → PDF 다운로드 버튼 클릭 → 견적서 PDF 파일 생성 및 다운로드
   [확인만 필요] → 화면에서 견적서 내용 열람 종료
```

## ⚡ 기능 명세

### 1. MVP 핵심 기능

| ID         | 기능명                     | 설명                                                                             | MVP 필수 이유                                             | 관련 페이지                            |
| ---------- | -------------------------- | -------------------------------------------------------------------------------- | --------------------------------------------------------- | -------------------------------------- |
| **[F001]** | 견적서 목록 조회           | Notion 데이터베이스에서 발행자가 작성한 견적서 목록을 불러와 표시                | 발행자가 자신이 등록한 견적서를 관리하기 위한 핵심 진입점 | 견적서 목록 페이지                     |
| **[F002]** | 견적서 상세 조회           | Notion 데이터베이스의 견적서 항목, 금액, 발행일 등 상세 내용을 화면에 표시       | 견적서 정보를 확인하는 핵심 가치 제공 기능                | 견적서 상세 페이지, 견적서 공유 페이지 |
| **[F003]** | 고유 링크 기반 견적서 열람 | 클라이언트가 로그인 없이 고유 링크로 접속해 해당 견적서만 조회                   | 클라이언트 비회원 접근이라는 핵심 사용 시나리오           | 견적서 공유 페이지                     |
| **[F004]** | 견적서 PDF 다운로드        | 브라우저에서 클라이언트 사이드 라이브러리로 견적서 화면을 PDF로 변환 후 다운로드 | 견적서를 파일로 보관/전달하려는 핵심 니즈 충족            | 견적서 상세 페이지, 견적서 공유 페이지 |

| **[F010]** | 기본 인증                  | 발행자(관리자) 회원가입/로그인/로그아웃만 제공                           | 발행자별 견적서 목록 구분을 위한 최소 인증              | 로그인 페이지, 회원가입 페이지 |

## 📄 페이지별 상세 기능

### 로그인 페이지

> **구현 기능:** `F010` | **메뉴 위치:** 인증 (비로그인 시) > 로그인

| 항목            | 내용                                                                                                       |
| --------------- | ---------------------------------------------------------------------------------------------------------- |
| **역할**        | 발행자(관리자) 인증 전용 페이지                                                                            |
| **진입 경로**   | 비로그인 상태에서 서비스 접속 시 자동 이동, 또는 회원가입 페이지에서 로그인 링크 클릭                      |
| **사용자 행동** | 이메일과 비밀번호를 입력해 로그인 시도                                                                     |
| **주요 기능**   | • 이메일/비밀번호 입력 폼<br>• 로그인 정보 유효성 검사<br>• 회원가입 페이지 이동 링크<br>• **로그인** 버튼 |
| **다음 이동**   | 성공 → 견적서 목록 페이지, 실패 → 에러 메시지 표시                                                         |

---

### 회원가입 페이지

> **구현 기능:** `F010` | **메뉴 위치:** 인증 (비로그인 시) > 회원가입

| 항목            | 내용                                                                                                                |
| --------------- | ------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 발행자(관리자) 신규 계정 생성 전용 페이지                                                                           |
| **진입 경로**   | 로그인 페이지에서 회원가입 링크 클릭                                                                                |
| **사용자 행동** | 이메일, 비밀번호, 비밀번호 확인을 입력해 계정 생성                                                                  |
| **주요 기능**   | • 이메일/비밀번호/비밀번호 확인 입력 폼<br>• 입력값 유효성 검사<br>• 로그인 페이지 이동 링크<br>• **회원가입** 버튼 |
| **다음 이동**   | 성공 → 로그인 페이지, 실패 → 에러 메시지 표시                                                                       |

---

### 견적서 목록 페이지

> **구현 기능:** `F001`, `F011` | **인증:** 발행자 로그인 필요

| 항목            | 내용                                                                                                                                                                                      |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 발행자가 자신이 등록한 견적서 전체를 한눈에 확인하는 메인 페이지                                                                                                                          |
| **진입 경로**   | 로그인 성공 후 자동 이동, 또는 발행자 메뉴에서 견적서 목록 클릭                                                                                                                           |
| **사용자 행동** | 견적서 목록을 확인하고, 동기화 버튼으로 Notion 최신 데이터를 불러오며, 특정 견적서를 클릭해 상세로 이동                                                                                   |
| **주요 기능**   | • Notion 데이터베이스 연동 견적서 목록 표시 (견적서명, 클라이언트명, 발행일, 금액 등)<br>• Notion 데이터 동기화 버튼<br>• 견적서 항목 클릭 시 상세 페이지 이동<br>• 동기화 진행 상태 표시 |
| **다음 이동**   | 견적서 클릭 → 견적서 상세 페이지                                                                                                                                                          |

---

### 견적서 상세 페이지

> **구현 기능:** `F002`, `F004`, `F012` | **인증:** 발행자 로그인 필요

| 항목            | 내용                                                                                                                             |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 발행자가 특정 견적서의 상세 내용을 확인하고 클라이언트 공유를 준비하는 페이지                                                    |
| **진입 경로**   | 견적서 목록 페이지에서 견적서 항목 클릭                                                                                          |
| **사용자 행동** | 견적서 상세 내용을 확인하고, 공유 링크를 생성/복사하며, 필요시 PDF로 다운로드                                                    |
| **주요 기능**   | • 견적서 항목별 내역, 수량, 단가, 합계 등 상세 표시<br>• 공유 링크 생성 및 복사<br>• 공유 링크 재발급<br>• **PDF 다운로드** 버튼 |
| **다음 이동**   | 공유 링크 복사 후 견적서 목록 페이지로 돌아가거나 페이지에 머무름                                                                |

---

### 견적서 공유 페이지

> **구현 기능:** `F002`, `F003`, `F004` | **메뉴 위치:** 클라이언트 메뉴 > 견적서 공유 페이지

| 항목            | 내용                                                                                                                                                                         |
| --------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 클라이언트가 로그인 없이 견적서 내용을 확인하는 전용 열람 페이지                                                                                                             |
| **진입 경로**   | 발행자에게 전달받은 고유 링크로 직접 접속 (별도 로그인 불필요)                                                                                                               |
| **사용자 행동** | 견적서 상세 내용을 확인하고, 필요한 경우 PDF로 다운로드                                                                                                                      |
| **주요 기능**   | • 고유 링크에 해당하는 견적서만 조회 가능<br>• 견적서 항목별 내역, 수량, 단가, 합계 등 상세 표시<br>• **PDF 다운로드** 버튼<br>• 잘못된/만료된 링크 접근 시 안내 메시지 표시 |
| **다음 이동**   | PDF 다운로드 완료 → 페이지에 머무름, 잘못된 링크 → 안내 메시지 표시                                                                                                          |

## 🗄️ 데이터 모델

### User (발행자 계정)

| 필드             | 설명                          | 타입/관계 |
| ---------------- | ----------------------------- | --------- |
| id               | 고유 식별자                   | UUID      |
| email            | 로그인 이메일                 | string    |
| password         | 암호화된 비밀번호             | string    |
| notionDatabaseId | 연결된 Notion 데이터베이스 ID | string    |
| createdAt        | 계정 생성 일시                | datetime  |

### Quote (견적서 캐시)

| 필드         | 설명                      | 타입/관계 |
| ------------ | ------------------------- | --------- |
| id           | 고유 식별자               | UUID      |
| userId       | 견적서를 등록한 발행자    | → User.id |
| notionPageId | 원본 Notion 페이지 ID     | string    |
| title        | 견적서명                  | string    |
| clientName   | 클라이언트명              | string    |
| totalAmount  | 견적 합계 금액            | number    |
| issuedAt     | 발행일                    | date      |
| syncedAt     | 마지막 Notion 동기화 일시 | datetime  |

### ShareLink (공유 링크)

| 필드      | 설명                        | 타입/관계  |
| --------- | --------------------------- | ---------- |
| id        | 고유 식별자                 | UUID       |
| quoteId   | 연결된 견적서               | → Quote.id |
| token     | 클라이언트 접근용 고유 토큰 | string     |
| isActive  | 링크 활성화 여부            | boolean    |
| createdAt | 링크 생성 일시              | datetime   |

## 🛠️ 기술 스택 (최신 버전)

### 🎨 프론트엔드 프레임워크

- **Next.js 15.5.3** (App Router) - React 풀스택 프레임워크
- **TypeScript 5** - 타입 안전성 보장
- **React 19.1.0** - UI 라이브러리 (최신 동시성 기능)

### 🎨 스타일링 & UI

- **TailwindCSS v4** (설정파일 없는 새로운 엔진) - 유틸리티 CSS 프레임워크
- **shadcn/ui** (new-york style) - 고품질 React 컴포넌트 라이브러리
- **Lucide React** - 아이콘 라이브러리

### 📝 폼 & 검증

- **React Hook Form 7.x** - 폼 상태 관리
- **Zod** - 스키마 검증 라이브러리

### 🔗 노션 연동

- **@notionhq/client** - Notion API 공식 SDK (견적서 데이터 읽기 전용 조회)

### 📄 PDF 변환 (클라이언트 사이드)

- **html2canvas** - 견적서 화면을 캔버스 이미지로 변환

### 📦 패키지 관리

- **npm** - 의존성 관리
